# nettoyage-canape-cergy.fr

Static local lead-generation website for validating demand around:

`nettoyage canapé Cergy`

## Purpose

This site is an MVP for a local SEO lead-generation model.

The goal is to rank for local service searches, capture calls/SMS/forms, and prove that the traffic can become sellable leads for local service providers.

## Live Site

https://nettoyage-canape-cergy.fr

## Lead Channels

- Phone: `07 56 80 43 67`
- SMS: `07 56 80 43 67`
- Formspree form endpoint

## Stack

- Static HTML
- CSS
- Small tracking script
- Cloudflare Pages
- GitHub Actions deployment

No CMS, database, dynamic rendering, or runtime page generation is required for the MVP.
